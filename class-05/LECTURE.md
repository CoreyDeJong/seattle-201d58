## Concept Review

- Terminal Commands
  - ls: list
  - pwd: print working directory
  - mkdir: makes a new directory / folder
  - touch: creates a new file
  - cp: copies a file / directory using a filname and a relative to a new location
  - mv: moves a file
  - rm: removes a file
  - rm -r: removes a directory

- git commands:
  - status
  - add
  - commit 
  - push

- HTML
